Vocab


  • Sprite - A graphics object that can be moved on top of or behind a background
  • Stage - The background of a project, performs functions through scripting
  • Costumes - one out of possibly many "frames" or alternate appearances of a sprite; Sprites can change their look to any of its costumes They can be named, edited, created, and deleted, but every sprite must have at least one costume
  • Sounds - Scratch's built in sound creator and modifier
  • Iterate - Repeating a step-by-step procedure several times; also used to refer to single execution of that repetitive procedure
  • Pseudocode - a way for developers to express algorithmic ideas without getting bogged down in syntactic details
  • Pair Program - software development technique in which two programmers work together at one workstation
  • Integrated Development Environment - Provides a developer with a way to create a program, run
  • See Inside - to enter a project select “see inside”; usually a blue button in top right corner
  • Camera - Press the camera button to take a picture. The picture will load then you can press the button to return to the picturing taking mode.
  • Allow - give permission to a certain task; allow microphone, allow access to camera, etc
  • Save - enables program to be a file and cannot be deleted after being erased
  • Import - To convert a file into the format required by the application being used
  • Select - process of highlighting text or picking an object
  • Delete - erases a file or document from the computer system you are using
  • Info - gives procedures, data, etc. on the project or file
  • Event - a message from one thread to another on scratch
  • Boolean Expressions - Evaluates to either true or false; used in the conditional of an if-structure
  • If then - structure that evaluates a Boolean expression once
  • Executes - To run a program or a single instruction
  • Bug - Part of a program that causes an error or undesired output
  • Debugging - The process of figuring out why code doesn't behave as expected and eliminating
  • Central Processing Unit Core - The part of the computer that executes one instruction at a time
